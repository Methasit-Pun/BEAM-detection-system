# Audio Alert Files

Place your audio alert files here.

## Required File

- `violation_alert.wav` - Audio alert for violations

## Audio Format

- **Format:** WAV (recommended) or MP3
- **Sample Rate:** 44100 Hz or 48000 Hz
- **Channels:** Mono or Stereo
- **Duration:** 1-3 seconds recommended
- **Volume:** Pre-normalized to avoid distortion

## Creating Alert Sounds

You can create alert sounds using:
- Online tools (freesound.org, zapsplat.com)
- Audio editing software (Audacity)
- Text-to-speech services

## Example Alert Messages

- "Violation detected"
- "Two riders detected"
- "Safety alert"
- Simple beep/buzzer sound
